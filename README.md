# Magic
## 命名規則

**型名(クラス、構造体、列挙体)**  
パスカルケース  
名詞  
例：GameObject  

**変数名**  
小文字スネーク  
名詞  
メンバ変数はアンダースコア終わり  
例：color_state  

**定数名**  
大文字スネーク  
名詞  
例：LEMON_NUM  

**関数名**  
パスカルケース  
動詞  
例：LemonCreate  

**フォルダ名**  
パスカル  

**Material**  
パスカル  
アタッチするゲームオブジェクト名  


## 作業の仕方

Workフォルダの中に自分の名前があるのでそこで作業を行ってください。  

作業が完了したら衝突していないか確認するためfork元のmasterをpullして確認してください。  
衝突していたら衝突しているところを自分で直してからpushしてください。  
確認できたら自分のリモートにpushし、pull requestを行ってください。  
これが一連の流れです。


## 注意点  
**Unityのバーション**  
5.1.1f1のバージョンで作業を行ってください  
ゲームオブジェクトは必ずプレハブ化してください。
